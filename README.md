# Homebrew Tap for streb

This is the official [Homebrew](https://brew.sh) tap for **streb** — a CLI tool that bootstraps Claude Code development environments with a single command.

## Installation

```bash
brew install stefantrockel/streb/streb
```

Or add the tap first:

```bash
brew tap stefantrockel/streb
brew install streb
```

## What is streb?

streb automatically sets up everything you need for productive Claude Code development:

- **Claude Code CLI** with AI-Hub API configuration
- **Beads** issue tracker for multi-session work
- **Production-ready templates** — slash commands, agents, hooks, and security rules

One command. Zero manual configuration.

→ **Learn more at [strebcli.dev](https://strebcli.dev)**

## Updating

```bash
brew upgrade streb
```
